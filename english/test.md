# git 명령어

- git clone 주소 : 원격 github 주소에서 로컬pc로 땅기기
- git add . : 변경 내용 저장
- git commit -m "변경 내용" : 로컬 피시에 최신 버전으로 확정
- git push origin master : 원격 서버에 업로드

# command 명령어

- ls : 폴더내용 리스팅
- cd 폴더명 : 폴더 접근
